## 系统软件管理

因为其实是ArchLinux的发行版，所以使用了pacman。
pacman的用法比较明确，但要注意的是
每次记得先`sudo pacman -Syy`一下。

顺道设置一下下国内源ovo
```
sudo pacman-mirrors -i -c China -m rank
sudo panman -Syy
pacman -S gcc
pacman -S vim
pacman -S make
pacman -S cmake
```
